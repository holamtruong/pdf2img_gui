# PDF TO IMAGE TOOL
- Version 1.0
![](https://github.com/holamtruong/pdf2img_gui/blob/master/screen.png?raw=true)

## Set up environment
### Create virtual environment:
    python -m venv venv
### Activate virtual environment (windows):
    venv\Scripts\active.bat
    
## How to run
### Install environment:
   Run: install.bat
### Run app:
   Run: main.bat
