# PDF to IMAGE TOOL
![](https://raw.githubusercontent.com/holamtruong/pdf2img_gui/master/screen.png?token=AEERSI3IB7ZP2TGGJW2NG727GPVFY)

## Set up environment
### Create virtual environment:
    python -m venv venv
### Activate virtual environment (windows):
    venv\Scripts\active.bat
    
## How to run
### Install environment:
   Run: install.bat
### Run app:
   Run: main.bat
